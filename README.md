# README

Code Workspace Extern project

## Requirement

- Node.js > 8.0
- make
- git (for update)

## Installation

in the folder you want to install the tool, run commond

```sh
git clone https://github.com/bagaking/code-workspace-ex.git .code-workspace-ex && /bin/bash -c ./.code-workspace-ex/install.sh
```
